# Repositório de SQL

Este repositório reúne todos os **scripts SQL** que estou estudando e desenvolvendo ao longo do tempo.  
O objetivo é registrar minha evolução, praticar diferentes técnicas e criar uma base de consultas úteis para futuros projetos.


Aqui estão scripts que envolvem:
- Criação e manipulação de tabelas  
- Consultas com **JOIN**, **GROUP BY** e **subqueries**  
- **CTEs**, **funções de janela** e **views**  
- Projetos de **data cleaning** e **análise de dados em SQL**

Cada pasta contém um conjunto de arquivos `.sql` com comentários explicando a lógica de cada script.
